# Sections Directory

This directory contains the main section components for the portfolio website.

## Planned Sections:
- `Hero/` - Hero section with animated introduction
- `About/` - About section with scroll-based reveals
- `Work/` - Work/projects showcase with interactive animations
- `Metrics/` - Metrics/stats section with counters
- `Contact/` - Contact section with form animations

Each section should be self-contained with its own animations and styles.
